## Built with

**HTML + CSS + Vanilla JavaScript** (no frameworks)
- **CSS Grid** for the main layout and gallery
- **Flexbox** for alignment inside the header and components
- **CSS variables** (custom properties) for the color palette in `:root`
- **Pseudo-elements** (`::after`) for the `LEARN MORE` highlight
- **Data attribute toggle** (`data-open="true/false"`) to show/hide the mobile menu
